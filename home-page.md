## Table

Em HTML
```html
<table border=1px>
  <thead>
    <tr>
      <th>This is a header</th>
      <th>This is a header too</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>This is a cell</td>
      <td>This is a cell too</td>
    </tr>
    <tr>
      <td>This is a cell on the second line</td>
      <td>This is a cell on the second line 2</td>
    </tr>
  </tbody>
</table>
```
